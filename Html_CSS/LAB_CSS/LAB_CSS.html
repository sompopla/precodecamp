1.Tooltips

Tooltips คือ กล่องข้อความ สำหรับอธิบายรายละเอียดของข้อความ ที่มีขนาดสั้นหรือคลุมเครือยากต่อความเข้าใจ

2.Pagination

Pagination คือ การแบ่งหน้าของข้อมูลให้มีหลาย ๆ หน้า ทำให้หน้าเว็บไม่ยาวจนเกินไป

3.Navigation Bar

Navigation Bar คือ แถบ Bar ที่ใช้ในการ link เนื้อหาในเว็บจากหน้าหนึ่งไปยังอีกหน้าหนึ่ง

4.Dropdown

Dropdown คือ กล่องข้อความ หรือ List box ที่ให้ผู้ใช้สามารถเลือกส่วนที่เราต้องการจากรายชื่อนั้น เมื่อ Drop-Down ไม่ทำงาน ตัวโปรแกรมจะแสดงเพียงแค่รายชื่อเดียวแต่ถ้าตัวโปรแกรมทำงานโดยเรากดตรงสามเหลี่ยม ตัวโปรแกรมจะแสดง รายชื่อทั้งหมดที่อยู่ในนั้นและให้เราเลือกอย่างใดอย่างหนึ่ง
