1.Attribute

Tag Attribute
h กำหนดส่วนหัวของ html
a href : ระบุ URL ของหน้าที่ลิงก์ไป
type : ระบุประเภทชนิดของเอกสารที่ต้องการเชื่อมโยง
target : ระบุตำแหน่งที่จะเปิดหน้าต่างของหน้า html
rel : ระบุความสัมพันธ์ระหว่างหน้าต่างปัจจุบันและหน้าต่างที่เชื่อมโยง

hreflang : ระบุภาษาของเอกสารที่เชื่อมโยง
div aline เพื่อจัดตำแหน่งให้เป็นบล็อค ๆ
p กำหนดย่อหน้าของเอกสาร
ul กำหนดรายการ (สัญลักษณ์แสดงหัวข้อย่อย)
th abbr : ระบุเวอร์ชันย่อของเนื้อหาในเซลล์ส่วนหัว
colspan : ระบุจำนวนคอลัมน์ที่เซลล์ส่วนหัวควรครอบคลุม
headers : ระบุเซลล์ส่วนหัวอย่างน้อยหนึ่งเซลล์ที่เกี่ยวข้อง
rowspan : ระบุจำนวนแถวที่เซลล์ส่วนหัวควรขยาย
scope : ระบุว่าเซลล์ส่วนหัวเป็นส่วนหัวสำหรับคอลัมน์แถวหรือกลุ่มคอลัมน์หรือแถว
table width : กำหนดความกว้างของตาราง
frame : ระบุว่าส่วนใดของเส้นขอบภายนอกที่ควรมองเห็นได้
align : เพื่อจัดตำแหน่ง
bgcolor : กำหนดสีด้านหลังแบคกรานด์ของตาราง
border : กำหนดขนาดขอบของตาราง
rules : ระบุว่าส่วนใดของเส้นขอบภายในที่ควรมองเห็นได้
cellpadding : กำหนดขนาดช่องว่างระหว่างตารางกับข้อความ
cellspacing : กำหนดขนาดช่องว่างระหว่างข้อความ
summary : ระบุข้อมูลสรุปของเนื้อหาของตาราง
selection กำหนดพื้นที่ของเอกสารที่ ยกตัวอย่างเช่นบทส่วนหัวส่วนท้ายหรือพื้นที่อื่น ๆ ของเอกสาร
option แท็กกำหนดตัวเลือกในรายการที่เลือก
link แท็กกำหนดการเชื่อมโยงระหว่างเอกสารและทรัพยากรภายนอก

2.Difference Tag

Tag ความเหมือน ความต่าง
hr & br เป็น tag ที่ไม่ต้องมี tag ปิด (Empty Elements)
  • hr = เป็นคำสั่งขีดเส้นใต้เพื่อแบ่งขั้น
  • br = เป็นคำสั่งสำหรับขึ้นบรรทัดใหม่ คล้ายกับการกด Enter บน คีย์บอร์ด
ul & ol เป็นคำสั่งใช้สำหรับสร้างรายการเป็น list
  • ul = สร้างรายการแบบไม่มีลำดับบนลงล่าง
  • ol = สร้างรายการแบบมีการเรียงลำดับเป็นตัวเลข
div & span เป็น tag ที่เอาไว้จัดกลุ่มของข้อความ
  • div = เป็น tag ที่มีไว้แบ่งส่วนเนื้อหาของ HTML ออกจากกันเป็นส่วนๆ (block)
  • span = มักใช้จัดแต่ง ข้อความสั้นๆ (inline) ที่อยู่ใต้การกำหนดรูปแบบของ tag อื่นอยู่แล้ว เพื่อให้มีรูปแบบที่ต่างไปจากรูปแบบเหล่านั้น และจะไม่ขึ้นบรรทัดใหม่
checkbox & radio button เป็น tag เกี่ยวกับการเลือกตัวเลือกที่กำหนดไว้ให้เลือก
  • checkbox = เลือกรายการได้หลายหลากอันเป็นจุดสี่เหลี่ยมตัวเลือก
  • radio button = เลือกรายการได้แค่อันเดียวโดยเป็นจุดวงกลมตัวเลือก
script & style เป็นการกำหนดชนิดข้อมูลที่จะใช้ในเอกสารเหมือนกัน
  • script=ทำให้ html สามารถใช้ javascript ในเอกสาร html ได้
  • style=ใช้กำหนดสไตล์ของข้อมูลเอกสาร html

3.Input Type

Type Description
button กำหนดปุ่มคลิกได้ (ส่วนใหญ่ใช้กับจาวาสคริปต์เพื่อเปิดใช้งานสคริปต์)
checkbox กำหนดช่องทำเครื่องหมาย
color กำหนดตัวเลือกสี
date กำหนดควบคุมวันที่ (ปีเดือนและวัน (ไม่มีเวลา))
datetime-local ประเภทการป้อนข้อมูลวันที่และเวลาได้ถูกลบออกจากมาตรฐาน HTML ใช้วันที่และเวลาท้องถิ่นแทน
email กำหนดเขตข้อมูลที่อยู่อีเมล
file กำหนดข้อมูลไฟล์และเลือกปุ่ม "Browse ... " (สำหรับการอัปโหลดไฟล์)
hidden กำหนดช่องใส่ที่ซ่อนอยู่
image กำหนดภาพเป็นปุ่มส่ง
month กำหนดเดือนและปี Control (ไม่มีโซนเวลา)
number กำหนดช่องสำหรับป้อนตัวเลข
password กำหนดฟิลด์รหัสผ่าน (ตัวอักษรหน้ากาก)
radio กำหนดปุ่ม
range กำหนดควบคุมสำหรับการป้อนจำนวนที่มีค่าที่แน่นอนไม่ได้เป็นสิ่งที่สำคัญ (เช่นการควบคุมเลื่อน)
reset กำหนดปุ่มรีเซ็ต (รีเซ็ตค่าทุกรูปแบบไปเป็นค่าเริ่มต้น)
search กำหนดฟิลด์ข้อความสำหรับการป้อนสตริงการค้นหา
submit กำหนดปุ่มส่ง
tel กำหนดฟิลด์สำหรับป้อนหมายเลขโทรศัพท์
text ค่าเริ่มต้น. กำหนดฟิลด์ข้อความบรรทัดเดียว (กว้างเริ่มต้นคือ 20 ตัวอักษร)
time กำหนดควบคุมสำหรับการป้อนเวลา (ไม่มีโซนเวลา)
url กำหนดฟิลด์สำหรับป้อน URL ที่
week กำหนดสัปดาห์และปี Control (ไม่มีโซนเวลา)
